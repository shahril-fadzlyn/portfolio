"U-Tiup" Clone Project

A responsive "U-Tiup" clone website, built with HTML and CSS

Project Overview

If you wondering, I intentionally use "U-Tiup" name to replaced the original website name to avoid any copyright issue.

This project is a responsive "U-Tiup" clone website that I created as a challenge to test my front-end development skills. After initially following a tutorial to build the project, I challenged myself to recreate it entirely from scratch without revisiting the tutorial.

Through this process, I gained hands-on experience in designing and implementing a responsive layout, improving my understanding of HTML, CSS, and JavaScript. I also honed my problem-solving skills by troubleshooting and resolving challenges independently.

This project highlights my ability to learn, apply, and adapt, making it an important step in my journey as a front-end developer.

Technologies Used

  • HTML5
  • CSS3
  • JavaScript
  • Responsive Design
  • CSS Grid & Flexbox

Key Features

  • Responsive layout that adapts to any screen size
  • Interactive elements
